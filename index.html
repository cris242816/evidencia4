🚀 Git Local vs Git Remoto

Descubre las diferencias fundamentales entre repositorios locales y remotos

📱 Git Local

  • 💾 Almacenamiento: En tu computadora personal
  • ⚡ Velocidad: Operaciones súper rápidas
  • 🔒 Privacidad: Solo tú tienes acceso
  • 💻 Trabajo offline: No necesita internet
  • 🎯 Control total: Cambios inmediatos

🔥 Comandos Locales

git init
git add .
git commit -m "mensaje"
git status
git log

🌐 Git Remoto

  • ☁️ Almacenamiento: En servidores (GitHub, GitLab)
  • 👥 Colaboración: Trabajo en equipo
  • 🔄 Sincronización: Actualización automática
  • 💎 Respaldo: Copia de seguridad en la nube
  • 🌍 Acceso global: Desde cualquier lugar

🚀 Comandos Remotos

git clone [url]
git push origin main
git pull
git fetch
git remote -v
